Was ist ein GiroCode? (SEPA-QR / EPC)

Ein GiroCode ist ein standardisierter SEPA-QR-Code (EPC) für Überweisungen. Er enthält u. a. Empfänger, IBAN, optional BIC, Betrag und Verwendungszweck. Banking-Apps können den Code scannen und die Überweisung automatisch ausfüllen.

Welche Daten stecken drin?

  • Empfänger (Name)
  • IBAN (Pflicht), BIC (optional)
  • Betrag (optional), Währung (EUR)
  • Verwendungszweck (bis ca. 140 Zeichen)

Vorteile

  • Schnell & fehlerarm: Scanner füllt Felder automatisch.
  • Standardisiert: EPC-Format ist in vielen Banking-Apps unterstützt.
  • Ideal für Rechnungen: Weniger Zahlendreher, schneller bezahlt.
